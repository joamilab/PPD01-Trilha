Antes de executar a aplicação:
1.Abra a classe JogoTrilha
2.Selecione o componente "tabuleiroPanelImagem"
3.Na propriedade "imagem" atualize o caminho para "[local onde a pasta do projeto está]\TelaTrilha\src\telatrilha\tabuleiro.png"
ou usando o botão de seleção selecione a imagem "src\telatrilha\tabuleiro.png" na pasta do projeto.

Como executar a aplicação:
1. Executar a classe Servidor.java;
2. Executar a classe Cliente.java duas vezes (uma vez para cada jogador).
